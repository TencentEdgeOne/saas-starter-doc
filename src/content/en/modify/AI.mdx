# AI Assistance

To develop projects more efficiently, here are some AI editing tools recommended:
- [Cursor](https://cursor.com/)
- [Windsurf](https://windsurf.com/editor)
- [GitHub Copilot](https://github.com/features/copilot)

You can quickly modify project code through dialogue, for example:

```
Modify landing page according to https://xxxx.xx.xx
```
