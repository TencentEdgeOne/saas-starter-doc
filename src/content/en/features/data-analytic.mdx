# Data Analytics
> Currently only supports Google Analytics, with more analytics tools to be supported in the future.

## Google Analytics
1. Enable [Google Analytics](https://developers.google.com/analytics) and obtain the tracking code
![](/ga.png)
2. Set the tracking code in your local environment variables
```
NEXT_PUBLIC_GA_ID=G-xxxxx
```
3. Wait 48 hours after deployment to see analytics data
