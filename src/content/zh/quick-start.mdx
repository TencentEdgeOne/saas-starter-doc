# 快速开始

### 前提条件
- Node.js 18+ 
- npm 或 yarn

### 安装依赖

1. **克隆仓库**
```bash
git clone https://github.com/your-username/saas-starter-template.git
cd saas-starter-template
```

2. **安装依赖项**
```bash
npm install
# 或
yarn install
```
### 本地开发
1. **启动开发服务器**
```bash
npm run dev
# 或
yarn dev
```

2. **打开浏览器**
访问 [http://localhost:3000](http://localhost:3000) 查看你的应用。
